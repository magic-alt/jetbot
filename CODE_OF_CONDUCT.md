# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short:

- Be respectful and welcoming.
- No harassment, discrimination, or personal attacks.
- Assume good intent; disagree on technical content, not on people.
- Report unacceptable behavior to the maintainers via the address listed in
  [`SECURITY.md`](SECURITY.md). Reports will be handled confidentially.

The full text is available at the link above and is incorporated here by
reference.
